**readModelFile** - *Read in a parsimony model file*

Description
--------------------

`readModelFile` Filler


Usage
--------------------
```
readModelFile(file, useambig = FALSE)
```

Arguments
-------------------

file
:   parimony model file.

useambig
:   use ambiguous naming as specified in the file?




Value
-------------------

A named vector containing the states of the model
